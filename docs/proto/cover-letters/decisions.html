CORE · Cover letters & outreach — design decisions

Status: IN FEEDBACK · Stage 1 confirmed 2026-07-05 (CORE questions incl.) · prototype at /proto/cover-letters/ · EXTENDS W2’s draft_cover (truth gate, structured output, quality gate kept).

Confirmed intent

The craft of sounding like you in a world drowning in machine-written mail. Starts with honesty about whether to write at all — RO says when a letter moves the needle and when it’s theater, write-anyway always yours. Voice is learned three ways and never stops updating: a skippable 60-second interview before the first letter, every edit anywhere, micro-questions when signal is thin. Letters are short, grounded (research lines cite their source — X2 briefs feed specificity), and dialed (warm/direct/formal, remembered; edits outrank the dial). Same comment system + 1-by-1 walker as the resume editor; truth gate unchanged. The outreach trio — cold note to a hiring manager, day-10 follow-up that adds one new thing, thank-you built around a moment you name — shares all machinery; every send is yours. Before any draft reaches you, she strips the AI tells — the goal isn’t sounding impressive, it’s sounding like you.

Locked decisions (incl. the three CORE questions)

#DecisionChoice
1World-class bar (CORE)Your voice + honest-about-when + deepest research (human triple combo). Beats the ChatGPT flood by not smelling like it; beats human writers by knowing when NOT to write; research specificity from X2 briefs, cited per line, never stalker-depth.
2Voice learning (CORE)All three channels, continuously updating (human combo): skippable upfront interview (3 questions, 60s — tone · one honest motivation sentence · optional writing sample) + every edit everywhere + micro-questions when stuck. The voice profile never stops tuning.
3Overridables (CORE)Tone dial (warm/direct/formal) per letter, remembered as default · edits outrank the dial, always · “skip the letter” and “write it anyway” both always available · same walker (1-by-1, editable, pausable) and comment system as the resume editor.
4ScopeLetters + the outreach trio: cold note to hiring manager (one real hook, no flattery) · day-10 follow-up (adds ONE new thing, never “checking in”) · thank-you (built around a moment the user names — reads like memory, not template). Same editor/gates; every send human-gated.
5The write-or-skip verdictPer company: does a letter move the needle here? Evidence-based (ATS behavior, stated hiring practice), honest “save your energy” when it’s theater. Anti-theater is a feature.
6AI-smell checkBefore any draft reaches the user, a tell-stripping pass (banned-phrase list + judge) removes machine-flavored writing; replaced only with profile-grounded specifics. Extends the voice blocklist in the quality gate.
7Length discipline~120 words for letters, ~60 for cold notes — brevity as respect. Word count shown honestly.

Data & guardrails

Voice profileNew first-class taste-model projection: interview answers + edit deltas + micro-question answers, confidence-weighted, user-inspectable (Settings), continuously updated. Powers letters, outreach, screening answers.
Truth gateW2 mechanics kept; research citations must trace to the X2 brief; user edits re-run the gate (J6 rule).
Human gateAll four artifact types are drafts; sends live in Apply/Send or the user’s own mail. Verified-before-shown + “not right?” (J8) apply.
